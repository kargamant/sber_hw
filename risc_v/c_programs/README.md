# Программа на C 

Программа на C включает в себя 2 функции, которые в итоге совместно покрывают как минимум по одной команде из каждого типа команд для RISC V.  
```C
int foo(int a, int b)
{
	if(a > b)
		return a + b;
	else 
		return a - b;
}

int bar_1(int a, int b)
{
	return foo(a, b) * 2;
}

int main()
{
	foo(1, 2);
	return 0;
}
```  

# Способ анализа  
Код компилировался с помощью компилятора clang-12 под rv32i архитектуру с уровнем оптимизации O0. В качестве ABI использовался [ilp32](https://riscv-non-isa.github.io/riscv-elf-psabi-doc).  Но поскольку просто код ассемблера выдаваемый clang-12 не содержит информации из линкера об адресах и не даёт посмотреть формат команды, то было принято решение сначала полностью провести этап компиляции и затем с помощью дизассемблера из исполняемого файла получить код с метками, адресами и форматами команд.  
```Makefile
all: asm object linker disasm
```
> Возможно способ не самый лучший, но рабочий. 

Все флаги и использованные опции можно посмотреть в [Makefile](https://github.com/kargamant/sber_hw/blob/main/risc_v/c_programs/Makefile).
