
# Слаки  

**Slack** - запас времени для определённого маршрута в схеме.  

Есть два вида слаков:  
**Setup slack** = Required_Time - Arrival_Time  
**Hold slack** = Arrival_Time - Required_Time  

**Required_Time** - требуемое время прибытия сигнала данных для корректной предустановки/удержания.  
**Arrival_Time** - реальное время прибытия сигнала данных.  

В соответствии с этими определениями, можно записать следующее:  
- **Setup slack** > 0 - сигнал придёт раньше, чем необходимо, время предустановки не нарушено.
- **Setup slack** = 0 - сигнал пришёл именно в момент времени предустановки.
- **Setup slack** < 0 - сигнал опоздал и нарушил время предустановки.  
- **Hold slack** > 0 - сигнал закончился позже, чем время удержания. Время удержания не нарушено.
- **Hold slack** = 0 - сигнал закончился именно в момент времени удержания.  
- **Hold slack** < 0 - сигнал закончился слишком рано и нарушил время удержания.
